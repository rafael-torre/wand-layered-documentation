## Alternative Paths

> **Guiding questions:**
>
> - What key branches or variations exist in this journey?
> - Does a returning user follow a different path than a new user?
> - Are there platform-specific differences (mobile vs. desktop) at the product behavior level?
> - What happens if the user abandons the journey midway and returns later?
>
> **Recommended length:** 1-2 paragraphs per alternative, or a second table if the alternative path is significantly different.

## Entry and Exit Points

> **Guiding questions:**
>
> - Where does the user come from before starting this journey? (another journey, external link, notification, direct access)
> - Where do they go after completing it? What is the natural next action?
> - What does success look like at the end of this journey — what has the user accomplished?
> - What does failure look like — what happens if the journey is not completed?
>
> **Recommended length:** 1 paragraph.
