# Tools — Layer 1: Product

AI skills and process guides that accelerate producing the artifacts in this layer. These are defined here as a backlog — each will be built out as a separate effort.

## AI Skills


| Skill                        | Type                                                                                                        | Status      |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- | ----------- |
| Feature Spec Drafting Skill  | AI skill: generates a first-draft feature spec from raw inputs (meeting notes, client briefs, chat threads) | Not started |
| User Journey Synthesis Skill | AI skill: synthesizes user journeys from feature specs and persona documents                                | Not started |
| Gap Analysis Skill           | AI skill: identifies contradictions and gaps across multiple raw inputs                                     | Not started |
