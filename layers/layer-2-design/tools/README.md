# Tools — Layer 2: Design

AI skills and process guides that accelerate producing the artifacts in this layer. These are defined here as a backlog — each will be built out as a separate effort.

---

## AI Skills


| Skill | Type | Status |
| ----- | ---- | ------ |
| Feature Design Spec Drafting Skill | AI skill: generates a first-draft feature design spec from a Layer 1 feature spec and Figma links, pre-filling states, interactions, and accessibility sections based on the feature's behavior | Not started |
| Design System Audit Skill | AI skill: reviews a set of feature design specs against the Design System Reference and flags inconsistencies (pattern deviations, undocumented component uses, missing accessibility references) | Not started |


---

## Process Guides


| Guide | Type | Status |
| ----- | ---- | ------ |
| Designer Handoff Checklist | Process guide: step-by-step checklist for marking designs as ready for dev — covers Figma readiness, design spec completeness, and accessibility review | Not started |
| Layer 2 Kickoff Checklist | Process guide: step-by-step checklist for starting Layer 2 on a new project — covers Design System Reference setup, Figma library organization, and first feature design spec | Not started |
